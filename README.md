# 🛒 E-Commerce ReactJS

A full-stack E-Commerce web application built using React, Node.js, Express, and Axios. Users can browse products, search for items, add products to their cart, update quantities, and proceed to checkout.

## 🚀 Features

- Product listing page
- Product search functionality
- Add products to cart
- Update cart quantity
- Remove items from cart
- Checkout page
- Order summary
- Responsive UI
- Backend API integration using Axios

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3
- Axios
- React Router

### Backend
- Node.js
- Express.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/arshiyamajumder30-lang/ecommerce-reactjs.git
cd ecommerce-reactjs
```

### Frontend Setup

```bash
cd ecommerce-project
npm install
npm run dev
```

### Backend Setup

```bash
cd ecommerce-backend
npm install
npm start
```
